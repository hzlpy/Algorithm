# 常见C语言字符串操作

标签： 算法 C语言

---

 1. char *strcpy(char *dst, char *src); //字符串拷贝函数 
 2. int strlen(char *src);//字符串长度函数，长度不包括结尾的'\0' 
 3. char *strcat(char *dst, char *src); //字符串连接函数
 4. char *strlwr(char *src); //字符串转换为小写函数 
 5. char *strupr(char *src);   //字符串转换为大写函数
